# Multimodal Agent Steering Surface

Chat-only agent products miss the way humans actually steer collaborators: pointing, circling, talking, interrupting, and reviewing shared context. This package turns that signal into a concrete UI pattern Vinay can reuse for ResumeSetGo-style reviews and agent workbenches.

Inspired by: https://x.com/mntruell/status/2062955210897801520

## What it contains
- `artifact-package/` — Obsidian package note mirror
- `prototype/` — dependency-free HTML prototype
- `infographic/` — workflow map
- `skill-draft/` — draft Hermes workflow
- `improvement-loop/` — recurring improvement tracker

## Quickstart
Open `prototype/agent-steering-surface.html` in a browser. Use `prototype/First Real Trial Packet.md` during the next real multimodal-agent workflow to capture source-backed input modes, output trace, reviewer notes, and the decision gate.
